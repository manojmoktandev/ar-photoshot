======= For RUN Node JS in your system =======
	
If you don't have Node Js in your  system
	1.Download Latest Node js  from https://nodejs.org/en/download/ according specs of your system.
	2.Do set up as given in documentation.

Node as installed already
	1. SYSTEMFOLDERPATH/npm install
		- after  full  installation of package
	2.SYSTEMFOLDERPATH/node server.js  for  viewer or  QA.
	3.SYSTEMFOLDERPATH/nodemon server.js  for developer  to apply  runtime changes.
